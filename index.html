Objectif
🎯 Hyrox Paris — 12 décembre 2026
Porte de Versailles. 22 semaines de prépa depuis mi-juillet.
Format : 8×1km course + 8 stations alternées.
Ordre : Ski → Sled push → Sled pull → Burpee broad jump → Row → Farmers → Sandbag lunges → Wall balls.
La course = plus de 50% du temps → priorité N°1.
📅 Les 3 blocs (périodisation)
BASE (S1-8) : volume aérobie + force lourde + technique stations légères.
SPÉCIFIQUE (S9-16) : compromised running, race pace, charges au-dessus du poids de course, demi-sims.
PEAK+TAPER (S17-22) : simulations complètes puis affûtage. Fraîcheur pour le jour J.

🏃 3 runs = le cœur de la prépa
Mardi (qualité) : intervalles VO2max ou tempo, alternés. Le 4×4min booste la VO2max 3× plus que le Z2.
Vendredi (spécifique) : course sous fatigue des stations = compromised running.
Samedi (long) : Z2 conversationnel, construit le moteur. Max 90 min.
😴 Récup progressive
Jeudi = easy dès S1 (marche/vélo facile + mobilité).
Dimanche = repos en bloc BASE, devient easy à partir de S9 quand ta tolérance au volume est construite.
Marqueurs pour ajouter du volume : sommeil bon + motivation ok + séances qui progressent. Sinon, garde le repos.
Les stations Hyrox
🧱 Wall balls — la pire sous fatigue
Une étude 2025 (Frontiers in Physiology) confirme : les stations paraissent plus dures que les runs, et les wall balls sont la station la plus éprouvante en fin de course. Entraîne-les toujours crampé. Poids course : 6kg (Open homme).
🛷 Sled — 2 stations sur 8
Sled push + sled pull = crucial. Turf lent, hanches basses, drive avec les jambes. Charges Open : push 152kg, pull 103kg (traîneau inclus). Si ta salle a un sled, priorise-le.
💪 Grip — farmers + ski + rameur
Le grip lâche souvent avant les muscles. Farmers carry 2×24kg (course), entraîne plus lourd. Ski erg = lats+triceps+core. Rameur = jambes+dos. Technique avant vitesse au début.
🔥 Compromised running = le secret
Enchaîner course → station → course, jambes cramées. C est ce qui fait la différence le jour J. Ne jamais séparer course et force en permanence, sinon le jour J sera un choc. Une sim complète coûte 5-7 jours de récup → une toutes les 2-3 semaines suffit.
Force & contraintes
🏋️ Force = maintien + puissance
2 séances/sem (jambes lundi, haut mercredi). En prépa Hyrox, la force préserve le muscle et développe la puissance sur les stations. Après décembre, tu repasses en prise de masse.
⚠️ Épaule — face pulls obligatoires
Zéro overhead lourd. Face pulls chaque jour de force (santé articulaire). Banded dislocations à l échauffement. Ski erg et wall balls OK (charge légère, mouvement contrôlé).
💪 Contraintes respectées
Pull-ups strictes uniquement (zéro kipping).
Pas de snatch barre — cleans OK, devils press (snatch haltère) OK.
Calisthénics (front lever) en entretien, pas en priorité pendant la prépa.
📉 Le taper (S20-22)
10-14 jours avant Paris : -40% de volume, on garde l intensité mais on réduit la quantité. Même une seule semaine d affûtage améliore la perf le jour J de façon mesurable. Objectif : arriver frais, pas cramé.
Nutrition
🎯 Jeûne intermittent 16:8
Fenêtre 12h-20h. Tu t entraînes l aprem = tu manges avant ET après la séance, dans la fenêtre. Zéro souci de perf ou récup.
Pendant le jeûne : café noir, thé, eau, électrolytes sans sucre.
🍚 Glucides remontés (vs prise de masse)
3 runs/sem = il faut du carburant. Les glucides remontent les jours durs (Lun/Mar/Ven/Sam ~210-220g). Jours easy (Jeu/Dim) glucides bas ~75g.
Jours durs : ~2200-2250 kcal · Jours easy : ~1750-1920 kcal.
⚡ Nutrition intra + course
Lun/Ven (séances Hyrox) : banane + miel + électrolytes pendant l effort = comme le jour J.
Samedi long run : teste ta stratégie nutrition course. Rien de nouveau le jour J.
